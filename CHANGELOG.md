# Changelog


## [2.5.1] - 2026-07-04

### ✨ Добавлено (Added)
- **История запросов**: создана отдельная страница истории (сохраняет до 50 последних успешных результатов ИИ).
- **Кнопка "Глазок"**: добавлена возможность показывать/скрывать API-ключ в настройках для удобства копирования.
- **Продвинутый Markdown парсер**: нейросеть теперь корректно форматирует списки (`- ` и `1. `) и абзацы, переводя их в чистый HTML (`<ul>`, `<li>`, `<br>`).

### 🛡 Безопасность и Валидация (Security)
- **Тест-драйв API-ключа**: добавлена предварительная валидация Mistral API ключа (через тестовый запрос) перед его сохранением в `chrome.storage`. 

### 🛠 Исправлено (Fixed)
- Исправлены ошибки строгой типизации TypeScript (`unknown` -> `any[]`) при работе с массивами данных из браузерного хранилища.

## [2.5] - 2026-07-04 (Текущая дата)

### ✨ Добавлено (Added)
- **Полная поддержка Google Docs**: реализован обход ограничений закрытого Canvas через чтение системного буфера обмена.
- **Интеграция в системное контекстное меню**: расширение теперь доступно по нажатию правой кнопки мыши (ПКМ).
- **Горячие клавиши (Hotkeys)**: добавлены официальные системные команды Chrome (Alt+R, Alt+Y, Alt+T) для быстрого вызова ИИ.
- **Сохранение Rich Text форматирования**: при изменении стиля или исправлении ошибок нейросеть сохраняет оригинальные HTML-теги (жирный шрифт, курсив, ссылки).
- **Инфраструктура тестирования**: внедрен фреймворк Jest для автоматизированного unit-тестирования модулей парсинга.

### 🛠 Исправлено (Fixed)
- Исправлен баг "залипания" всплывающего окна (Drag & Drop) при отпускании левой кнопки мыши.
- Устранена ошибка смешивания контекста: нейросеть больше не возвращает лишний текст вокруг выделенного слова.
- Исправлено поведение всплывающей панели в сложных `input` и `textarea` полях.

### ♻️ Изменено (Changed)
- Улучшена архитектура: логика парсинга ответов вынесена в отдельную утилиту `src/utils/textParser.ts`.
- Обновлен системный промпт модели `mistral-large-latest` для более строгого следования формату без лишних приветствий.
